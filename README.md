# 🚗 Uber Ride Analytics Dashboard

An interactive **Power BI dashboard** built to analyze Uber ride data, uncover trends, and generate actionable business insights across revenue, demand, and customer behavior.

---

## 📊 Overview

This project transforms raw ride data into meaningful insights through intuitive visualizations.

It focuses on:

* Ride demand trends over time
* Revenue distribution across vehicles and locations
* Customer and driver behavior
* Cancellation patterns and operational gaps

---

## ❓ Problem Statement

Ride-hailing platforms often struggle to:

* Identify peak booking periods
* Understand high cancellation rates
* Track revenue contribution by vehicle type
* Analyze performance across different locations

This dashboard addresses these challenges using data-driven analysis.

---

## 📌 Key Insights

* 🚕 Auto & Bike generate the highest revenue
* 📍 Certain locations dominate booking demand
* ❌ Cancellation trends highlight service inefficiencies
* 📅 Monthly patterns reveal peak and low-demand periods
